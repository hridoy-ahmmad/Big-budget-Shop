# BIG BUDGET

## Purpose
BIG BUDGET is a modern e-commerce product showcase website built with Next.js. Users can browse products, view product details, sign up, sign in, manage their profile, and explore summer sale offers.

## Live URL
https://big-budget-shop-wdsp.vercel.app

## Key Features
- Responsive homepage with summer sale hero section
- Product cards with image, category, price, rating, and details button
- All products page for browsing the full product collection
- Dynamic product details page
- User authentication with sign up, sign in, and sign out
- Protected profile page
- Profile update functionality
- Toast notifications for user actions
- Mobile-friendly navigation menu
- Summer care tips section
- Service feature section including secure payment, free shipping, easy returns, and eco-friendly support
